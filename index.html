Internship Documentation

Welcome to the official documentation of my internship project. This site presents a structured overview of all tasks, experiments, tools, and learnings gained throughout my internship, with a strong focus on Docker, virtual machines, networking, performance testing, and security tools.

ğŸ¯ Project Objective

The core aim of this internship is to evaluate the performance, flexibility, and efficiency of using Docker containers compared to traditional virtual machines. The focus is on:

  • How fast and lightweight containers are for lab setups
  • Resource consumption of VMs vs Docker
  • How containers affect workflow, scalability, and repeatability
  • Testing communication, network behavior, and performance
